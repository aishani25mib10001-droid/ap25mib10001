# HOSPITAL MANAGEMENT SYSTEM
<br>
<h2>OVERVIEW OF THE PROJECT</h2>
<br>
Hospital Management System (HMS) is an application in Pythonâ€“SQLite built for the digitalization and management of basic hospital operations. The system supports efficient management of patients, doctors, appointments, and medical records. It improves workflow, reduces manual errors, and enhances overall hospital administration efficiency.
<br>


--------------------------------------------------------------------------------------------------------------------------------------------------------------

<br>
<h2>FEATURES</h2>
<br>
<h4>Major Functional Modules</h4>
<br>

- **Patient Management**
  - Add, update, delete, search patient information  
  - Store diagnosis, treatment, and visit history

- **Doctor Management**
  - Manage doctor profiles, specialties, availability  
  - View assigned patients

- **Appointment Management**
  - Book, modify, cancel appointments  
  - Check doctor schedule and patient history

<h4>Additional Features</h4> 
 
- Secure login authentication  
- SQLite database storage  
- Interactive menu-driven interface  
- Error-handling for invalid inputs

<h2>INSTALLATION AND SETUP</h2>

### Prerequisites
- Python 3.8+  
- SQLite (built-in with Python)  

### Steps to Install
1. Clone the repository
2. Navigate to the project folder:
cd Hospital-Management-System
3. Run the main program:
python main.py



### Database Setup
- The database (`hospital.db`) will be auto-created on first run.  
- No manual setup required.  

## How to Run the Project
python main.py<br>
Use the on-screen menu options to navigate through modules.
